# list

List all log systems and their current levels.

## Usage

```
piri client admin log list
```

## Example

```bash
piri client admin log list
```

```json
{
  "piri/pdp": "info",
  "piri/storage": "info",
  "piri/retrieval": "debug",
  "piri/claims": "info",
  "piri/replicator": "warn"
}
```
